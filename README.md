# LearnPress LMS Demo Website 🚀

## 📖 Description
This is a fully responsive static website demo for **LearnPress LMS** - a modern online education platform inspired by popular WordPress LMS themes/plugins. It showcases how users can build skills through online courses, with sections for browsing categories, featured courses, blog articles, and user authentication.

**Purpose**: Educational landing page and course catalog to promote online learning programs like Art & Design, Development, Marketing, and more.

## ✨ Features
- **Home Page**: Hero banner, top categories (8+ categories), featured courses, stats (25K+ students, 899 courses), promotional sections.
- **Courses Page**: Full course grid/list view, search bar, pagination, course details (duration, students, free/paid).
- **Blog Page**: Article listings with images, dates, search, grid/list toggle, pagination.
- **Login/Register**: Dual form page with email/username/password fields and "Remember me".
- **Other Pages**: Contact, FAQ.
- **Navigation**: Responsive navbar with dropdowns (Pages: Contact/FAQ), mobile offcanvas/accordion menu.
- **Footer**: Links, contact info, social icons (Facebook, Instagram, X/Twitter, YouTube, Pinterest).
- **Responsive Design**: Mobile-first, works on all devices (media queries, Bootstrap components).
- **Interactive Elements**: Search bars, view toggles (grid/list), breadcrumbs, hover effects.

## 🛠️ Technologies Used
- **HTML5** - Semantic structure.
- **CSS3** - Custom styles (Flexbox/Grid, animations), Google Fonts (Exo, Jost).
- **Bootstrap 5** - Responsive framework (navbar, dropdowns, offcanvas, pagination, breadcrumbs).
- **Font Awesome 7** - Icons (search, clock, users, social, etc.).
- **JavaScript** - Bootstrap JS for interactivity (no custom scripts needed).

## 🚀 How to Run
1. Open `index.html` in any modern web browser (Chrome, Firefox, etc.).
   - Windows: Double-click or run `start index.html` in terminal.
2. No server or build tools required - fully static!
3. Navigate via navbar to explore pages.
